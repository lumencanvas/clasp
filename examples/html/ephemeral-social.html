channels: append #ch=roomname to the URL for a private channel. everyone with the same URL is in the same room.

how it works: posts use CLASP SET (persistent, replayed on connect). reactions and live signaling use CLASP emit (ephemeral). live video is WebRTC P2P via STUN — works for most home/mobile NAT. enterprise/symmetric NAT requires a TURN server.

image size: photos are compressed to ~60KB before sending as base64 in the CLASP payload. images expire when the post does.

relay: relay.clasp.to (public). self-host with lumencanvas/clasp.
